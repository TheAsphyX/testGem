# Calcolatrice Semplice

Questo progetto implementa una calcolatrice web semplice utilizzando HTML, CSS e JavaScript.

## Funzionalità

La calcolatrice supporta le seguenti operazioni di base:
- Addizione (+)
- Sottrazione (-)
- Moltiplicazione (*)
- Divisione (/)
- Calcolo del risultato (TOTALISSIMO)
- Cancellazione completa (AC)
- Supporto per numeri decimali (.)

## Tecnologie Utilizzate

- **HTML5**: Per la struttura della pagina web e degli elementi della calcolatrice.
- **CSS3**: Per lo stile e il layout della calcolatrice, rendendola intuitiva e visivamente gradevole.
- **JavaScript**: Per la logica di funzionamento della calcolatrice, gestendo l'input dell'utente, eseguendo i calcoli e aggiornando il display.

## Come Avviare l'Applicazione

Per avviare la calcolatrice, segui questi semplici passaggi:

1.  **Naviga nella directory del progetto**:
    Apri il tuo file explorer o terminale e vai alla directory in cui si trovano i file del progetto.

2.  **Apri `index.html`**:
    Fai doppio clic sul file `index.html`. Questo aprirà l'applicazione nel tuo browser web predefinito.

    In alternativa, puoi fare clic destro su `index.html` e scegliere "Apri con" e selezionare il tuo browser preferito.

## Utilizzo

-   Fai clic sui pulsanti numerici per inserire i numeri.
-   Fai clic sui pulsanti degli operatori (+, -, \*, /) per selezionare l'operazione desiderata.
-   Usa il pulsante "." per aggiungere un decimale.
-   Fai clic su "TOTALISSIMO" per visualizzare il risultato del calcolo.
-   Fai clic su "AC" per cancellare il display e resettare la calcolatrice.
